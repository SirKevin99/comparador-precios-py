# comparador-precios-py

Aplicación desarrollada en **TypeScript** que permite comparar precios de productos de manera sencilla.

## Requisitos previos

- **Node.js** v20.19+ o v22.12+ (recomendado v22 LTS)
- **npm** 

Verificar versiones:

```bash
node -v
npm -v
```

## Clonar repositorio

```bash
git clone https://github.com/SirKevin99/comparador-precios-py.git
cd comparador-precios-py
```

### Instalar dependencias del BackEnd

```bash
cd backend
npm install
```

#### Crear el archivo de configuracióm

- .env

### Instalar dependencias del FrontEnd

```bash
cd ../frontend
npm install
```

#### Crear el archivo de configuracióm

- .env

### Levantar el proyecto

Levantar el Back y el Front en terminales separadas.

```bash
cd backend
npm run dev
```

```bash
cd frontend
npm run dev
```
